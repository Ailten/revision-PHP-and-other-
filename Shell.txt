700  // full private (only acess for owner of file).
444  // only readable (for all).
666  // read and write (for all), not execute.
644  // r+w for owner, read only for other (recomanded for file).
755  // owner can do all, r+x for other (reomanded for folder).
750  // group users can read + execute (other can do nothing).
711  // other can execute, without see the contend.
